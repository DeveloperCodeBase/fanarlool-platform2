## Fanar Lool QC Platform Demo (Vite + React + TypeScript)

**پلتفرم نمایشی** برای جلسه مدیرعامل کارخانه فنر لول ایران. شامل ماژول‌های QC مرکزی (ماژول ۳)، OEE (ماژول ۵) و انرژی (ماژول ۶) با داده کاملاً نمایشی و دو زبانه فارسی/انگلیسی (RTL/LTR).

### پیش‌نیازها
- Node.js 18+

### نصب و اجرا
```bash
npm install
npm run dev
```
سایت به صورت پیش‌فرض با زبان فارسی (RTL) باز می‌شود. `npm run build` برای خروجی production.

### مسیرها
- `/` لندینگ با اسکرول نرم به سکشن‌های قهرمان، ماژول‌ها، معماری، امنیت، قیمت‌گذاری نمایشی و تماس
- `/platform` نمای جزئیات QC مرکزی (مدل‌ها، Recipe، ردیابی و نمودارها)
- `/demo` داشبورد یکپارچه با کنترل‌های سراسری کارخانه/بازه/شیفت/محصول
- `/company` معرفی شرکت و عضویت‌ها
- `/contact` کارت‌های تماس + فرم نمایشی

### لوگو
فایل نمونه در `public/logo.svg` قرار دارد. برای استفاده از لوگوی واقعی، فقط فایل `public/logo.svg` را جایگزین کنید؛ کامپوننت `Logo` به صورت خودکار آن را نمایش می‌دهد.

### داده نمایشی و اتصال واقعی
- تمام اعداد برچسب «داده نمایشی» دارند.
- منبع داده: `src/data/mockGenerator.ts` که کارخانه فنر لول ایران و ۳ کارخانه عضو (A/B/C) را شبیه‌سازی می‌کند: خطوط تولید، رویدادهای OEE، بازرسی QC، انرژی، هشدارها و تعرفه‌ها.
- برای اتصال واقعی:
  - **QC/OEE**: تعبیه OPC UA / Modbus / MQTT Gateway در بخش Edge/Gateway. می‌توانید به جای `generateMockData`، درخواست‌های REST/WebSocket به سرویس جمع‌آوری داده متصل کنید.
  - **انرژی**: داده کنتورهای برق/گاز/هوای فشرده (Power meter کلاس عمومی، کنتور گاز، فلومتر هوا) را در همان ساختار EnergySample ارسال کنید.
  - نقطه شروع: ساخت یک service در `src/context/AppContext.tsx` که داده زنده را فچ کرده و جایگزین `mockGenerator` کند.

### استقرار
- build: `npm run build` → خروجی در `dist/`
- پیش‌نمایش: `npm run preview`
- میزبانی پیشنهادی: ابر خصوصی یا آن‌پرمیس در کارخانه فنر لول ایران یا دیتاسنتر پارک علم و فناوری سمنان (SSL + احراز هویت).

### ساختار پوشه‌ها
- `src/pages` صفحات اصلی (Landing, Platform, Demo, Company, Contact)
- `src/components` اجزای مشترک (Navbar, Footer, Logo, LanguageToggle) و `ui/` (Card, Badge, Tabs, Table, Modal, Drawer, Sidebar, Skeleton, KpiCard)
- `src/data/mockGenerator.ts` تولید داده نمایشی و فیلتر بر اساس انتخاب‌ها
- `src/i18n` تنظیمات دو زبانه
- `src/utils/format.ts` قالب‌بندی اعداد و درصد بر اساس زبان

### نکته‌های توسعه
- استایل با Tailwind و گرادیان‌های ملایم + شیشه‌ای
- سوئیچ زبان در ناوبار: جهت صفحه و تایپوگرافی RTL/LTR را عوض می‌کند.
- همه نمودارها به انتخاب کارخانه/بازه/شیفت/محصول واکنش نشان می‌دهند.
